Project Idea

Scan the laptop of user using camera and and display all its related accessories like battery, ram, mouse, headphones, charger etc along with the prices using augumented reality. Here the accessories includes any type of components that can be used along with laptop(mouse, headphones) or replaced for betterment(like RAM, hardisk etc). The price information of the accessories can be retrieved from amazon, ebay etc. On clicking the selected accessory, the user should be able to navigate to corresponding amazon/ ebay etc website for purchase.
